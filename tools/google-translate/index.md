# Google Translate

**Kort fortalt:** Rask oversettelse, kamera/lyd-støtte og mange språk.

## Hva er det best til?
- Raske oversettelser
- Kamera og stemme
- Mange språk

## Hvordan komme i gang
- Åpne translate.google.com
- Velg språkpar
- Skriv, snakk eller last opp bilde

## Gratis vs. betalt
- Gratis

## Prompter / eksempler
- Oversett dette avsnittet til engelsk og behold tallformat
